🎓 Sistema de Análise de Notas (POO & Rich)

Este projeto consiste em uma classe Python que utiliza **Programação Orientada a Objetos** para gerenciar e analisar notas de alunos, exibindo um relatório visual diretamente no terminal.

## ✨ Destaques do Projeto

* **POO (Programação Orientada a Objetos):** Uso de classes e métodos para encapsular a lógica de cálculo.
* **Argumentos Variáveis (`*args`):** Permite inserir qualquer quantidade de notas para um único aluno.
* **Interface Estilizada:** Utiliza a biblioteca `rich` para criar painéis coloridos, textos sublinhados e cores dinâmicas baseadas no desempenho (média).

## 🛠️ Pré-requisitos

Para rodar este script, você precisará instalar a biblioteca `rich`:

```bash
pip install rich
