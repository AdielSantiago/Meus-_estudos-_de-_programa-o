⚡ Calculadora de Consumo Elétrico (Python)

Este é um script simples em Python desenvolvido para calcular o custo estimado mensal de energia elétrica de diferentes equipamentos domésticos.

## 🚀 Funcionalidades

* Interface limpa via terminal (limpa a tela automaticamente).
* Cálculo baseado em Watts, tempo de uso diário e tarifa padrão.
* Geração de um relatório final com o custo individual e o total acumulado.
* Suporte multi-plataforma (Windows, Linux e macOS).
